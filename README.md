# Advyay AI Website

Welcome to the Advyay AI website repository. This project is built with Next.js and showcases various AI-powered solutions.

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Run the development server
   ```bash
   npm run dev
   ```

3. Build for production
   ```bash
   npm run build
   ```

4. Start the production server
   ```bash
   npm start
   ```

## Project Structure

- `app/` - Next.js pages and API routes
- `components/` - Reusable UI components
- `public/` - Static assets
- `styles/` - Global styles and CSS

## Deployment

This site is deployed to Vercel automatically on pushes to the `main` branch.
